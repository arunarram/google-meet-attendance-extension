# google-meet-attendance-extension

### Enabling the extension

1. Download the repository and extract it
   
   <img width="698" alt="downloading code" src="https://user-images.githubusercontent.com/69054617/114155325-fc9b7280-993e-11eb-9595-897940cd9eb5.PNG">

2. Turn on the DeveloperMode on the browser(chrome)
  <img width="957" alt="developerMode_before" src="https://user-images.githubusercontent.com/69054617/114008692-22157700-9880-11eb-8af7-ab51566c15c2.PNG">

  <img width="963" alt="developerMode_After" src="https://user-images.githubusercontent.com/69054617/114008764-2e99cf80-9880-11eb-81dd-3aa4c025bf9c.PNG">

  
3. Press load unpacked and select the folder containing the repository files
  <img width="596" alt="loadUnpacked" src="https://user-images.githubusercontent.com/69054617/114008816-39ecfb00-9880-11eb-9593-41d5be916133.PNG">

  
4. Now you can use the extension and make sure Meet Attendance Extension is ON
  <img width="308" alt="Meet Attendance Extension" src="https://user-images.githubusercontent.com/69054617/114008888-47a28080-9880-11eb-91b9-38e62fa1c8a7.PNG">

  
5. Pin the Extension (so that meet icon is visible from the toolbar)    
  <img width="254" alt="extension on the bar" src="https://user-images.githubusercontent.com/69054617/114008919-50935200-9880-11eb-9538-3f742c1f1f0a.PNG">
